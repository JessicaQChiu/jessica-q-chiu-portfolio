## About The Project

Portfolio website created to learn React, TypeScript, and Tailwind CSS. Inspired by neubrutalism and Windows 98 interface designs. No AI used. All original art by me. 

### Built With
* Next.js
* React
* TypeScript
* Tailwind CSS
* CSS
